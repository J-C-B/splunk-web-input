================================================
Overview
================================================

This app provides a mechanism for pulling information from websites.



================================================
Configuring Splunk
================================================

This app exposes a new input type that can be configured in the Splunk Manager. To configure it, create a new "Website Content" input in the Manager under Data inputs.
